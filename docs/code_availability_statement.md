# Code Availability statement for the manuscript

The code used to construct the training corpus, including source normalization, text chunking, dialogue synthesis prompts, tool-trace augmentation, quality filtering and deterministic data splitting, together with the FractureAgent prompt templates, tool schemas, deterministic safety gate, ms-swift/Swift QLoRA configuration and launchers, and evaluation scripts, is available in the accompanying GitHub repository: https://github.com/hangcao87/FractureAgent. The repository includes synthetic fixtures and does not redistribute downloaded source documents or the curated training corpus; users must obtain open-access materials under the terms of their original providers. The trained FractureAgent model weights, LoRA checkpoints, private API credentials and non-public institutional materials are not included because they are being retained for ongoing intellectual-property protection, follow-on validation and grant-related work. The repository provides instructions for supplying a legally obtained local base model and reproducing the data-processing and ms-swift/Swift training workflow without access to the authors' checkpoint.

## Short version

Code, prompts, tool schemas, data-processing scripts, ms-swift/Swift training scripts and evaluation scripts are available at https://github.com/hangcao87/FractureAgent. The curated training corpus, trained model weights and checkpoints are not publicly released at this stage because they are being retained for ongoing research, intellectual-property protection and grant-related work. No patient-level data are distributed. Reasonable requests concerning derived data may be directed to hangcao87@163.com, subject to source licences and author approval.
